Latitude Vs Analysis

The goal of this analysis was to see if the temperature would indeed increase as we got closer to the equator. This was done by pulling data from the OpenWeatherAPI to build a dataset of over 500 cities

After building this dataset we used Python and Matplotlib to plot the data in various permutations. The key ones we will be covering are latitude against temperature, cloudiness, wind speed and humidity. This site provides the raw data and the visualizations created as part of the analysis.
